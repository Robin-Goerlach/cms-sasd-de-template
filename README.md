# CMS-SASD-DE Template

A custom Joomla template for **cms.sasd.de**, designed for **SASD GmbH**.

This template is based on a clean and maintainable child-template approach and adapts Joomla's frontend appearance to the SASD visual style with a calm **navy / petrol** color palette, modernized UI elements, and a professional layout suitable for a technical company website.

## Goals

The template was created to provide:

- a consistent SASD GmbH visual identity
- a professional and calm appearance
- a maintainable structure for future customization
- a solid basis for further CMS and branding work

## Features

- SASD-inspired navy and petrol color scheme
- improved styling for typography, buttons, cards, forms, and tables
- reusable CSS utility classes for branded content sections
- lightweight structure based on Joomla child-template principles
- suitable as a starting point for future SASD website evolution

## Project Scope

This repository contains the Joomla template package and related source files for the SASD website frontend theme.

Depending on further development, the repository may later include:

- template source refinements
- screenshots and preview assets
- installation notes
- branding adjustments
- release packages

## Installation

Install the template in Joomla via the extension manager, then assign it as the default site template in the Joomla backend.

## Intended Use

This template is intended primarily for the SASD GmbH web presence and related internal or customer-facing CMS projects.
